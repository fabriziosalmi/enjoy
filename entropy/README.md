## How We Calculate

### Entropy Factors (Bad - increases disorder)

| Factor | Description |
|--------|-------------|
| Activity Spread | How concentrated is activity in time? |
| Karma Inequality | Is karma distributed fairly? |
| Bounty Ratio | Unclaimed opportunities |
| Level Entropy | Progress vs potential |
| Streak Entropy | Continuity of engagement |

### Anti-Entropy Factors (Good - creates order)

| Factor | Description |
|--------|-------------|
| Achievement Order | Accomplishments create structure |
| PR Order | Contributions build the system |
| Community Order | More souls = more coherence |

### The Formula

```
Net Entropy = (Sum of Entropy Factors) - (Sum of Anti-Entropy Factors)
Order Score = 100 - Net Entropy
```

## Fighting Entropy

Want to increase the Order Score? Here's how:

1. **Contribute regularly** - Consistency beats intensity
2. **Help new players** - Community growth = anti-entropy

4. **Maintain streaks** - Continuity is order
5. **Share karma** - Equality reduces disorder
