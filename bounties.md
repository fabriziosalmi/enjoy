## 🟢 Active

### 🐛 Bug Hunter · +50 karma
**Difficulty:** Easy

Find and report any bug in the game. Open an [Issue](https://github.com/fabriziosalmi/enjoy/issues/new?template=bug-report.yml) with:
- Clear description of the problem
- Steps to reproduce
- Screenshot if possible

## ⚡ How It Works

1. **Choose** an active bounty
2. **Complete** the required task
3. **Cite** the bounty in your PR/Issue: `Bounty: [name]`
4. **Receive** the bonus karma on merge/close
